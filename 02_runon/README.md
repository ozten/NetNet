# RunOn

Our NetNet foundation model is has a very limited context.
Given an initial prompt, we generate a long-form response
by iteratively calling the model with the previous response.

## Setup

    source .venv/bin/activate
    cd 02_runon    

    # Create model.pt
    python runon.py
